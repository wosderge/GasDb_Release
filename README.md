# GASDB

## 概要
グーグルスプレッドシートを用いたDBです。  
NCMBに近い操作感を意識しています。

## 利用方法
詳しくは下記note記事に記載しております。   
「ここにnote記事のアドレスを置く」


ざっと説明すると  
・Scriptフォルダ内のデータをunityプロジェクトに追加  
・appscript.gsの内容をスプレッドシートのapp scriptにコピペ  
となります。

また、当プロジェクトはUniTaskを使用しておりますので、UniTask環境の導入も必要となります。
